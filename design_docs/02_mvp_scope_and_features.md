# MVP Scope – Features

## Included in MVP (v0)

### 1. Users & Family
- Individual profiles
- Family grouping (CH)
- Aggregated family metrics and rewards

### 2. Routine Builder (Time-Based & Expiring)
- Routines grouped by moment of day:
  - Morning / Midday / Evening / Night
- Supports:
  - Products (meds, supplements, skincare, hair care)
  - Habits (walk, water, etc.)
- Each routine item supports:
  - Start date
  - Expiration date OR duration (e.g. 14 days)
  - Optional next-version pointer

### 3. Exercise Planning (Goal-Aligned)
- Workout plans aligned to individual goals
- Simple progression logic
- Post-workout check-in (RPE, notes)

### 4. Habits
- Binary or numeric habits
- Daily check-ins
- Streak tracking

### 5. Weekly Body Composition Input
- Manual form
- Metrics configurable per user
- Trend visualization (basic)

### 6. Daily AM Brief & PM Recap
- AM: plan + motivation + reminder
- PM: adherence summary + adjustment

### 7. Dashboard + Gamification
- Daily score (0–100)
- Streaks (habit, routine, global)
- Family momentum score
- Rewards unlock system

## Deferred
